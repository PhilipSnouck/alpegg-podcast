# Alpegg Podcast

Private family podcast project (Dutch): this repo holds the static project page
(`index.html`, zero build step) and planning docs — podcast production itself
happens outside the repo.

**Run locally:** open `index.html` in a browser, or `npx serve -p 3456 .`
(see `.claude/launch.json`) and visit http://localhost:3456.

**Pointers:**
- README.md = usage/explanations
- DEVELOPER.md = architecture
- ROADMAP.md = task list (format spec:
  `C:\Users\p.snouckaert\Personal repos\roadmap-dashboard\ROADMAP_TEMPLATE.md`
  — read before editing)
